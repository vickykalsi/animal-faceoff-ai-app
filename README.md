# Animals Faceoff AI App
a simple app which takes user input and generates output by utilizing google's gemini ai api

## Features
- a very simplified UI
- ability to enter your desired opponents
- ability to clear previous faceoff analysis

## Tech Stack
- Frontend : EJS, CSS and JS
- Backend : Node.js and Express
- API : Google's Gemini API

## Installation and Setup
- git clone https://github.com/vickykalsi/animal-faceoff-ai-app.git
- cd "animal-faceoff-ai-app"
- npm install
- generate your own API key from https://aistudio.google.com/api-keys and save it   in .env
- npm start
- Browse http://localhost:3000 from your browser
